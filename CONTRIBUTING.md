# Contributing to GhostTrace

First off, thank you for considering contributing to GhostTrace! It's people like you that make GhostTrace such a great tool.

## How Can I Contribute?

### Reporting Bugs
Before creating bug reports, please check the existing issues to see if the problem has already been reported. When creating a bug report, please include as many details as possible.

### Suggesting Enhancements
Enhancement suggestions are tracked as GitHub issues. Describe the enhancement you'd like to see and why it would be useful.

### Pull Requests
1. Fork the repository and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Issue that pull request!

## Style Guide
- Use PEP 8 for Python code.
- Keep documentation up to date.

## Questions?
Feel free to open an issue with the tag `question`.
